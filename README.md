# SleepSense AI

A Flask-based machine learning web application for predicting sleep disorders using health and lifestyle data.

## Features
- Sleep disorder prediction using Random Forest
- User authentication system
- Admin dashboard
- Prediction history tracking
- Interactive web interface

## Tech Stack
- Python
- Flask
- Scikit-Learn
- SQLite
- HTML/CSS/JavaScript

## Installation

```bash
pip install -r requirements.txt
python app.py
